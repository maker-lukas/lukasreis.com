# Getting Started

you'll need a few things before getting started

- Text editor (I'll be using Code OSS)
- Default Minecraft resource pack



This  is the folder stucture you'll need for this project and its aleardy made so that you can add more textures

```
project root/
├── pack.mcmeta
├── pack.png
│
└── assets/
    └── minecraft/
        ├── models/
        │   └── item/
        │       └── sword/
        │           └── diamond/
        │               └── fire_aspect.json
        │
        ├── textures/
        │   └── item/
        │       └── sword/
        │           └── diamond/
        │               ├── diamond_sword.png
        │               ├── fire_aspect.png
        │               └── fire_aspect.png.mcmeta
        │
        └── items/
                └── diamond_sword.json
```
